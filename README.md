# Student Skill Gap Analyzer

A Flask-based web application that analyzes student skills, finds missing skills, recommends courses, and provides a career-based learning roadmap.

## Features

- Career selection
- Skill gap analysis
- Skill score calculation
- Course recommendations
- Personalized learning roadmap

## Technologies

- Python
- Flask
- HTML
- JSON

## How to Run

Install Flask:

pip install flask

Run:

python app.py

Open:

http://127.0.0.1:5000
